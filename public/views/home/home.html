Welcome to Alumnance!

Alumnance is an easy way to take alumni attendance at special events.

To get started, upload a CSV file of all the alumni in eTapestry (or whatever your current alumni management system is).

NOTE: Uploading a CSV file erases any previous alumni records from Alumnance. That usually doesn't matter since Alumnance is just for taking attendance, not archiving alumni records. The only time it really concerns you is while you're taking attendance during an event. You don't want to upload the CSV file then, because it will erase the attendance records you've entered!

If you're using eTapestry:

  1. Sign in to eTapestry (click here to go to the sign-in page).
  2. Under Reports on the menu at the top of the page, click System.
  3. Find the All Alumni report on the page (you may have to scroll down).
  4. Click Run Report just underneath it.
  5. Make sure that the selected query is All Alumni - A. Change it to that if it isn't set to it already.
  6. Change the Report Format to CSV File - Download.
  7. Click Submit.
  8. Wait for the report to be generated and downloaded.
  9. Log out of eTapestry unless you need it for something else.
  10. Back in Alumnance (on this very page), click the button at the bottom to select a file.
  11. Select the file that you just downloaded from eTapestry. If you're on Windows, it's probably in your Downloads folder, named something like AllAlumni.
  12. Wait for the file to upload. After a moment, you should get a message saying the upload succeeded.

If you're using other alumni management software:

Alumnance will work with any alumni management software that can export CSV files. The CSV file should have four columns:

  1. Full Name
  2. Maiden Name
  3. School(s) Attended
  4. Class Of
A header row is required. The column names don't matter, but the order does.
