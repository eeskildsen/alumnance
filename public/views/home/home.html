Welcome to Alumnance!

Alumnance is an easy way to take alumni attendance at special events.

Upload CSV
To get started, upload a CSV file containing all alumni in your alumni management system. Show me how
The CSV file should contain four columns:
  1. Full Name
  2. Maiden Name
  3. School(s) Attended
  4. Class Of
A header row is required. The column names don't matter, but the order does.
